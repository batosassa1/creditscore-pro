1. Introduction

CreditScore Pro est une application web complète d'évaluation de crédit utilisant l'intelligence artificielle. Elle permet de :

  • Évaluer l'éligibilité au crédit d'un utilisateur
  • Calculer un score de crédit (0-100)
  • Déterminer le montant maximum empruntable
  • Fournir des recommandations personnalisées
  • Visualiser les résultats avec des graphiques interactifs
Technologie : Flask (Python) + HTML/CSS/JavaScript + Machine Learning

2. Installation

Prérequis

  • Python 3.11 ou 3.12 (le modèle ML exige scikit-learn 1.3.2, indisponible pour Python 3.13)
  • pip (gestionnaire de paquets Python)

Étapes d'Installation

# 1. Créer l'environnement virtuel (une seule fois)
py -3.12 -m venv .venv

# 2. Installer les dépendances (versions exactes)
.venv\Scripts\pip install -r requirements.txt

# 3. Lancer l'application
.venv\Scripts\python app.py

L'application sera accessible à : http://localhost:5000

Astuce : Vous pouvez aussi double-cliquer sur le fichier start.bat pour lancer automatiquement l'application.

4. Formulaire d'Évaluation

Le formulaire est divisé en 3 étapes :

Étape 1 : Informations Personnelles

  • Âge : Entre 18 et 75 ans
  • Situation professionnelle : CDI, CDD, Indépendant, etc.
  • Ancienneté : Nombre d'années dans l'emploi actuel
  • Situation familiale : Célibataire, Marié(e), etc.
  • Personnes à charge : Nombre de dépendants
  • Niveau d'études : Du Bac au Doctorat

Étape 2 : Situation Financière

  • Revenu mensuel net : En F CFA
  • Charges mensuelles : Dettes actuelles en F CFA
  • Épargne totale : Montant total épargné en F CFA
  • Historique de crédit : Excellent, Bon, Moyen, Difficile
  • Prêts en cours : Nombre de prêts actifs
  • Situation logement : Propriétaire, Locataire, etc.

Étape 3 : Prêt Souhaité

  • Montant du prêt : Montant souhaité en F CFA
  • Objectif : Immobilier, Véhicule, Personnel, etc.
  • Durée : En mois (ex: 60 mois = 5 ans)
Important : Tous les champs sont obligatoires. La validation se fait en temps réel.

5. Résultats

Après soumission, vous obtenez :

Score de Crédit (0-100)

  • 0-40 : Non éligible ❌
  • 40-60 : Éligible sous conditions ⚠️
  • 60-100 : Éligible ✅

Montants Disponibles

  • Montant Minimum : Pour un petit projet
  • Montant Recommandé : Optimal pour votre profil
  • Montant Maximum : Capacité maximale d'emprunt
  • Mensualité estimée : Paiement mensuel

Graphique Radar

  • Situation Professionnelle

Recommandations

Conseils personnalisés pour améliorer votre score ou optimiser votre prêt.

7. Intégration du Modèle ML

Prérequis

  • Modèle entraîné avec scikit-learn
  • Méthode predict_proba() disponible
  • Format .pkl ou .joblib

Étapes

# 1. Sauvegarder votre modèle
import joblib
joblib.dump(model, 'model/model.pkl')

# 2. Placer le fichier dans :
# credit_scoring_app/model/model.pkl

# 3. Redémarrer l'application

Pour plus de détails, consultez le fichier MODEL_INTEGRATION.md dans le dossier du projet.
